1.Podstawowe informacje.

    Po uruchomieniu aplikacji i wpisaniu w przeglądarce localhost:6767 wyświetli się poniższa instrukcja:

        "Aby skorzystać z zaimplementowanej funkcjonalności  jako endpoint należy podać -
                    "/SEARCH?str=(tutaj podajemy poszukiwanego rodzaju działaności)";


2. Przykładowe dane.

    Po wpisaniu "http://localhost:6767/str?=szkola - api wyświetli nam stronę ze znalezionymi wywnikami wyszukiwania dla słowa "szkola" wraz z przyciskami , które po kliknięciu zwraca w nowym oknie dane w formacie VCF.
